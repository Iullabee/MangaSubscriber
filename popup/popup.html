The Browser Action is your entry point :

The Icon

  • On the icon is displayed the number of mangas that have at least one chapter you have yet to read (no matter how many unread chapters a manga has, it will be counted as one).
  • When you click it, it will bring up the popup menu.
  • It is unavailable on firefox for android !

The PopUp consists of different parts :

The first part is the Menu

  • add the manga to your list. It can also be : (already in your list), or (not a manga on a supported website)
  • open/close the options panel.
  • open/close the console panel.
  • open/close the help panel (this one).
  • version of the extension. close any opened panel.

The second part filters the list and apply actions to all visible mangas

  • hide all mangas in the list whose title doesn't contain what is typed inside.
  • toggle the display of mangas with unread chapters.
  • toggle the display of fully read mangas.
  • refresh the list.
  • The subsequent buttons apply to all visible ( = not filtered) mangas : open selected chapter in a new tab, update, mark all chapters as 'read', / toggle update on/off, remove from the list

The third (and last) part is the list

  • Each line represents a manga (name, number of unread chapters, and the list of chapters, followed by the open selected chapter at the end).
  • Hovering over a manga will extend it to show additional options applying to that manga only : remove from the list, select the primary website, update, mark all chapters as 'read', / toggle update on/off

The Options Panel gives you the following possibilities :

The Options

  • check all sites : lets you choose whether to check only a manga's preferred website when looking for new chapters, or all available websites.
  • navigation bar : will let you navigate between chapters (first, previous, next and last) when reading a subscribed manga.
  • automatically update : will update the list every X hours.
  • export list to file : will export your list to a local json file.
  • import mangas list from text : will import data copy/pasted in the field.
  • import mangas list from file : will open a webpage to import data saved in a file.
  • export list to pastebin : will anonymously export your list to pastebin.
  • import list from pastebin : will import data previously exported to pastebin.
